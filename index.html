Welcome to the Udacity Storefront

The following routes are given:

    #### Users
    - Index `/users` [GET] [token required]
    - Create `/users` [POST] ========== payload: { firstname, lastname, username, password }
    - Read `/users/:id` [GET] [token required]
    - Update `/users/:id` [PUT] [token required]
    - Delete `/users/:id` [DELETE] [token required]
    - Auth `/users/auth` [POST] ========== payload: { username, password }
    
    #### Products
    - Index `/products` [GET]
    - Create `/products` [POST] [token required] ========== payload: { name, price, category(optional) }
    - Read `/products/:id` [GET]
    - Update `/products/:id` [PUT] [token required] ========== payload: { name, price, category(optional) }
    - Delete `/products/:id` [DELETE] [token required]

    #### Orders
    - Index `/orders` [GET] [token required]
    - Create `/orders` [POST] [token required]
    - Read `/orders/:id` [GET] [token required]
    - Delete `/orders/:id` [DELETE] [token required]
    - GetCurent `/orders/current/:user_id` [GET] [token required]
    - GetByActive `/orders/actived/:user_id` [GET] [token required]
    - GetByComplete `/orders/completed/:user_id` [GET] [token required]
